# Landing Page "Polla Futbolera" - Mundial 2026

Esta es una Bridge Page (Landing Page de aterrizaje) diseñada para captar tráfico de Google Ads para el Mundial 2026, enfocada en el concepto de "Polla Futbolera" (Quinielas).

## 🚀 Características Premium

- **Diseño Moderno:** Interfaz limpia con estética "Light Mode", acentos en verde vibrante y efectos de *Glassmorphism*.
- **Optimización para Ads:** Copywriting diseñado para evitar bloqueos en Google Ads (sin menciones a Web3, Crypto o Apuestas), enfocado en entretenimiento y competencia entre amigos.
- **Automatización:** Enfocado en la eliminación de Excels manuales y cálculos automáticos en tiempo real.
- **UI de Alta Calidad:** Construido con Next.js 16, Tailwind CSS v4 y Shadcn UI.
- **Micro-interacciones:** Animaciones sutiles y estados de hover para una experiencia de usuario fluida.

## 🛠️ Tecnologías

- **Framework:** Next.js (App Router)
- **Estilos:** Tailwind CSS
- **Componentes:** Shadcn UI (Radix UI)
- **Iconos:** Lucide React

## 📦 Instalación y Desarrollo

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```
